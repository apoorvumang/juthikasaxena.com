# juthikasaxena.com

Personal site for Juthika Saxena.

Plain static HTML/CSS/JS — no build step.

## Files

- `index.html` — About Me
- `writings.html` — Writings index
- `styles.css`, `script.js`, `assets/`

## Preview locally

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.

## Deployment

Hosted on GitHub Pages with `juthikasaxena.com` as the custom domain. Pushing to `main` deploys automatically.
